# Gol
Para compilar um .jar execute o build.sh passando o caminho do arquivo scala-library.jar
```
./build.sh CAMINHO
```
# Requisitos:
  * scala - Scala programming language
  * fastjar - Jar creation utility
# Executar
Para executar faça:
```
java -jar Gol.jar
```
# Limpar
Para limpar a compilação faça;
```
./build.sh limpar
```
